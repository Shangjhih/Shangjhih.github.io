联想电脑OEM镜像获取

发表于 2025-09-13
本文字数： 661 阅读时长 ≈ 1 分钟

联想笔记本在官网并未直接提供iSO镜像的下载位置，官方给出的有有一个叫做Lenovo USB Recovery Creator tool for Windows的工具，点击此链接就可以下载。

关键在于iSO的获取，官方提供的工具可以下载iSO并且可以作为镜像写入工具，将其下载的镜像写入到U盘内作为启动器。

重点在于如何获取到iSO,这里需要一个联想账号，自行注册，注册流程就不再说，常规账号的注册流程。

随后登录账号。

点击此链接进入联想服务界面,此处的国家必须是日本，语言可以自己选，切换语言并不会影响所在国家服务

https://pcsupport.lenovo.com/jp/zc/lenovorecovery

查找自己的设备并获取SN码，该码是设备的唯一标识符，携带该设备的所有信息。请正确获取该识别码（该识别码是一串8位的字符，由大写字母和数字组成）

随后在中间的搜索框中输入你获取到的SN码，并点击右侧的搜索图标，随后即可看到你的设备信息，请核对具体的版本。
国家和地区务必选择香港！！！语言选择简体中文

请务必接受许可条款，不然无法进行下一步

随便编一个名字并提交

至此，你的账号就拥有了下载该设备镜像的权限，接下来请下载前面提到的Lenovo USB Recovery Creator tool for Windows 下载到本地并登录进你已经获得下载权限的账户，登陆成功后则会提示你现在已经可以下载,点击确认并将其导航到合适的文件夹开始下载iSO.

当文件下载到本地时，您可以使用这个联想官方的恢复工具来制作恢复介质，或者将此文件复制到您的PE中使用。
