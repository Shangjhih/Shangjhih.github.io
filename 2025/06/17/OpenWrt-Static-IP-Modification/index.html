OpenWrt Static IP Modification

发表于 2025-06-17 分类于 OpenWrt ， Network ， Static ip
本文字数： 2.4k 阅读时长 ≈ 4 分钟

OpenWrt修改静态IP需要使用基础的linux命令进入后台对其进行修改，或者在网页端的后台中进行，本次我们大概了解一下通过ssh后台进行修改的具体步骤。

大概的步骤为：通过ssh连接进入终端，cd切换到网络配置文件的绝对路径（/etc/config/network），使用vim命令对network文件进行修改（修改option gateway和option ipaddr两个参数，其中gateway参数设置为OpenWrt的上级主路由地址，ipaddr参数修改未主路由网段下的任意IP（用于后台登陆））。

SSH后台登录

在powershell使用以上命令进行登录或ssh客户端添加OpenWrt的登录信息

命令解析：

ssh ————————————-使用ssh协议进行登录

root————————————用户名

@—————————————–指定目标地址的符号

192.168.*.*————————-设备内网地址

第一次输入该命令后，powershell会提示是否保存该地址的指纹信息至ssh配置信息内，此时输入yes并回车，在地址输入正确之后会提示你输入密码（在输入密码时窗口不会显示输入的密码）在密码输入之后直接回车进入到控制台，提示如下：

此时就代表已经成功进入后台，可随意对系统进行修改

修改配置文件

vim———————————————————–Linux中的文本编辑软件

/etc/config/network——————————–配置文件的绝对路径

直接在空白处输入该命令，就会进入到编辑状态：

此时使用方向键将光标移至option gateway所在行，按下按键i进入编辑模式（vim使用特性），进入编辑模式后及可对地址进行修改，修改后对option ipaddr的参数也进行修改，修改为与网关同网段的地址，可以根据自己的想法及逆行设置，保证最后一位的数字为2-225中的任何一位，且要保证不能与已分配的地址相同。（内网地址不可冲突！！！）

在地址修改完成后按esc退出vim的编辑模式
