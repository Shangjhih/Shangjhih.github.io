Play Integrity修复(Root)

发表于 2025-11-05
本文字数： 3k 阅读时长 ≈ 5 分钟

你需要知道的

在使用Sora或者很多需要依赖Google play integrity的软件时，你的第三方ROM可能会导致Playintegrity检测无法通过，关于playintegrity是什么，请参见网页：

  • 谷歌突然出手，Android刷机又被戴上了紧箍咒—–知乎

  • 使用 Play Integrity API 检测存在风险的互动并防范滥用行为—-Google play管理帮助中心

  • Play 完整性和签名服务—–Android Developer

总结:play integrity检测不通过会影响部分软件的运行，必须将设备的play integrity安全等级提升到DEVICE及以上才能保证设备的完整性。

playintegrity的安全等级分为BASIC,DEVICE ,STRONG , 等级依次升高。DEVICE及以上即可获得Google Play的设备认证。

解决方案

注意：内容引用自xdaforums论坛的Rajarshi Khatua（原内容为英文），[原文章点此](Rajarshi Khatua)

本教程将帮助您成功通过传统和新版游戏完整性响应中的基本完整性检查、设备完整性检查和强完整性检查。请务必仔细按照每个步骤操作。

第一步：移除旧模块

开始之前，请移除之前安装的与 Play Integrity Fix 相关的任何模块，以避免冲突。

步骤二：下载所需组件

下载以下工具和模块：

  • Zygisk Module(choose one)

    • ReZygisk (Recommended)
    • NeoZygisk

  • Play Integrity Module (choose one):

    • PlayIntegrityFix (Recommended)
    • PlayIntegrityFork

  • Tricky Tools

    • TrickyStore
    • Tricky Addon – Update Target List

步骤 3：（可选）安装 KSU Web UI

仅当您使用 Magisk时才需要安装以下组件之一：

  • 肯特州立大学网页用户界面
  • WebUI X（替代方案）

第四步：重启设备

所有组件安装完毕后，重启手机以使更改生效。

步骤 5：配置 PlayIntegrityFix

  1. 打开PlayIntegrityFix。
  2. 关闭“*向 Play 商店伪装 SDK 版本” 选项。
  3. 点击“获取 pif.json”以检索配置文件。

步骤 6：配置 TrickyStore

  1. 打开TrickyStore。

  2. 点击操作按钮（右下角）。

  3. 点击菜单（⋮）并选择：

    • 全选
    • 取消选择不必要的
    • 设置有效密钥盒

步骤 7：设置安全补丁日期

  1. 再次点击菜单。
  2. 选择“设置安全补丁”。
  3. 点击“获取安全补丁日期”。
  4. 然后点击“保存”。

步骤 8：禁用 ROM 欺骗（重要）

如果您使用的是自定义 ROM，请务必禁用任何 ROM/GMS 欺骗选项，因为它们可能会干扰 Play Integrity 的结果。

步骤 9：验证强完整性

请使用以下应用程序检查您的完整性状态：

  • 游戏完整性 API 检查器

如果一切操作正确，你应该已经通过了：

  • 基本诚信
  • 设备完整性
  • 强大的诚信

适用于传统和新型响应格式。

重要提示

避免过于频繁地检查 Play 完整性。
仅在绝对必要时才进行检查。检查次数过多可能会引起 Google 的怀疑，并可能导致您的设备即使之前通过了完整性验证，现在也开始无法通过。

英文原文

This tutorial will help you successfully pass Basic, Device, and Strong Integrity checks in both Legacy and New Play Integrity responses. Make sure to follow each step carefully.

Step 1: Remove Old Modules

Before starting, remove any previously installed modules related to Play Integrity Fix to avoid conflicts.

Step 2: Download Required Components

Download the following tools and modules:

  • Zygisk Module

    (choose one):

    • ReZygisk (Recommended)
    • NeoZygisk

  • Play Integrity Module (choose one):

    • PlayIntegrityFix (Recommended)
    • PlayIntegrityFork

  • Tricky Tools:

    • TrickyStore

  • Tricky Addon – Update Target List

Step 3: (Optional) Install KSU Web UI

Only if you’re using Magisk install one of the following:

  • KSU Web UI
  • WebUI X (alternative)

Step 4: Reboot Your Device

Once all components are installed, reboot your phone to apply the changes.

Step 5: Configure PlayIntegrityFix

  1. Open PlayIntegrityFix.
  2. Turn off the option “Spoof SDK version to Play Store”.
  3. Tap Fetch pif.json to retrieve the config file.

Step 6: Configure TrickyStore

  1. Open TrickyStore.

  2. Tap the Action button (bottom right).

  3. Tap the menu (⋮)and select:

    • Select All
    • Deselect Unnecessary
    • Set Valid Keybox

Step 7: Set Security Patch Date

  1. Tap the menu again.
  2. Select Set Security Patch.
  3. Tap Get Security Patch Date.
  4. Then tap Save.

Step 8: Disable ROM Spoofing (Important)

If you’re using a custom ROM, make sure to disable any ROM/GMS spoofing options, as they can interfere with the Play Integrity results.

Step 9: Verify Strong Integrity

Use the following app to check your integrity status:

  • Play Integrity API Checker

If everything was done correctly, you should now pass:

  • Basic Integrity
  • Device Integrity
  • Strong Integrity

For both legacy and new response formats.

Important Note

Avoid checking Play Integrity too often.
Only do it when absolutely necessary. Too many checks can make Google suspicious and may cause your device to start failing integrity even if it was previously passing.
