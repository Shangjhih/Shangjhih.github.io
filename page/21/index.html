Win11 25H2跳过在线账户

发表于 2025-10-11 分类于 Win11 ， oobe
本文字数： 1.3k 阅读时长 ≈ 2 分钟

在Windows11 25H2版本的更新后，微软取消了以前使用oobe跳过联网登录在线账户的功能，尽量保证用户都使用微软账户登录使用。但是在初次购买笔记本测试的时候我们要避免联网激活，所以跳过登陆激活是极其必要的。这里我们将提供其它可用的跳过联网方法。该信息具有时效性，请查看本文的建立时间并比对您的实际情况参考使用。

注意：跳激活期间设备不能联网

已失效的方法

oobe命令

在25H2之前的版本中，可以使用快捷键shift + F10 快捷键调出CMD运行框，通过命令oobe\bypassnro 跳过联网验证（该命令会在联网环节调出我没有internet 窗口，允许用户使用本地账户登入Windows）。

  • 笔记本用户可以在激活前使用Shift + Fn + F10 组合键调出运行框
  • 台式组装机可以使用上文提到的Shift + F10

注意：输入命令并回车后系统会重启，随后才会在联网阶段出现【我没有internet连接】选项，通过此选项即可建立本地账户。

localonly命令

与上文同，在激活前的预备界面使用同样的方式打开CMD，但是输入命令,效果是一致的。
