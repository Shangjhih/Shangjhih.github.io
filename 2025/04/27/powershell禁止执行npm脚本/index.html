执行策略说明

  • Restricted：默认设置，不允许运行任何脚本
  • RemoteSigned：允许运行本地脚本，但从网络下载的脚本需要签名
  • Bypass：允许运行所有脚本（最不安全，但最方便）
  • AllSigned：所有脚本都需要数字签名才能运行

建议使用 RemoteSigned 策略，这样可以运行大多数本地脚本，同时保持一定的安全性。

另一种替代方法

如果您不想更改执行策略，也可以使用 CMD 命令提示符，而不是 PowerShell：
