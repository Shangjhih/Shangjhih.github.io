预定制Windows系统-AME wizard&Playbook

发表于 2025-08-14 分类于 Windows ， playbook
本文字数： 3.4k 阅读时长 ≈ 6 分钟

前情提要

本内容主要涉及AME Wizard软件的下载和Playbook配置文件的下载,以及文件下载后的使用方法。
核心在于Playbook文件的选择（下载链接已经贴出）和配置文件的修改项目选择。


预定制的Windows系统？什么是预定制？和Windows操作系统又有什么关系？

首先要明确的是，预定制Windows完全基于官方的系统，官方系统的来源无非就是购买设备时已经灌装的OEM系统，也就是商家预置的系统，另一种是你自己使用PE或微软官方提供的iSO镜像自行安装的系统。

OEM和微软官方系统有什么区别？

OEM:各家电脑品牌方都会购买Windows的授权并基于其硬件对微软官方的系统进小规模或大规模的修改，以满足自己的硬件或品牌需求。比如联想电脑会在OEM预装的系统中安装联想电脑管家以及各种控制中心。

微软官方：由微软官方提供的操作系统镜像，是最基础的Windows版本，没有任何第三方的软件按与定制文件。

什么是Playbook？

Playbook 是一个包含一系列系统修改的小文件。此 Playbook 中的更改由开源AME Wizard应用，该工具可以修改正在运行的 Windows 系统。这允许您将当前安装转换为 ReviOS，而无需删除系统并重新安装。

与之前的 ISO 版本一样，ReviOS Playbook 删除了过多的软件和不必要的系统组件，提升了隐私保护和系统性能，同时保持了兼容性。完整的更新列表可在此处查看。

Playbook 兼容 Windows 10 版本21H2 (19044)和22H2 (19045)（仅限 X64）以及 Windows 11 版本23H2 (22631)和24H2 (26100)（X64 和 ARM64）。

这涵盖标准 Windows 10、11 和 LTSC。它也适用于任何 Windows 版本，这意味着无论哪个版本，您都可以使用现有的 Windows 许可证密钥。

Playbook 通常与Windows 版本无关，这意味着 Playbook 可以在任何受支持的 Windows 版本上运行（请参阅此列表）。这意味着您无需担心 Playbook 版本与 Windows 版本的匹配问题。您也不必等待新的 Playbook 版本发布后再更新 Windows 安装。

根据经验法则，无论您想在哪个 Windows 版本上使用它，都应该始终使用最新的可用 Playbook 版本。



摘自ReviOS

功能介绍

Playbook文件是一个yaml文档，该文档中写的是对Windows系统修改的自定义参数，也就是说这个文件可以定义Windows中的功能，可以添加额外的功能（Windows11右键菜单取消多级列表、禁止自动更新…）以及删减掉不必要的功能（比如微软的遥测功能以及信息收集功能…）,Playbook文件需要搭配执行该配置的软件使用——AME Wizard

AME Wizard是一个用于修改Windows系统的一个开源工具，该工具可以执行Playbook文件的配置来根据预设的功能来对Windows的功能进行修改。但是有一个前提：

要将 AME 向导与指定的 Playbook 配合使用，您必须安装该 Playbook 支持的 Windows 版本。目前，AME 11支持所有稳定的 Windows 11 版本，AME 10支持 Windows 10 21H2 和 22H2。

在进行安装使用前请检查您的系统版本。如何检查？

win+r打开运行框，输入winver回车，检查自己的系统版本是否为以上提到的版本（win11全版本支持，win10仅支持21H2及更新版本）

各版本区别

Revi OS :

  • 类型：带预做优化的 Windows ISO（也有 Playbook 格式的配置）
  • 核心特点：
    • 内置禁用遥测、广告推送、自动更新调度器
    • 删除部分 UWP 应用，但保留核心功能（微软商店可选）
    • 调整服务启动类型（多数服务延迟启动，提升空闲性能）
    • 启用更多高级性能参数（TCP 优化、低延迟模式）
  • 安装后主要变化：
    • 系统占用较低，但比 Atlas 更日常化（注重稳定性）
    • 更适合长期桌面使用 + 游戏混合
    • 更新服务可选保留或精简，不像 AME 那么激进

Atlas OS:

  • 类型：偏向游戏调优的安装后执行脚本（Playbook）
  • 核心特点：
    • 禁用 Windows Telemetry、广告服务、OneDrive、Xbox Game Bar
    • 优化 GPU/CPU 电源策略，减少 DPC 延迟
    • 关闭动画、特效以降低输入延迟
    • 精简后台进程（网络发现、打印服务等）
  • 安装后主要变化：
    • 系统功能较完整（微软商店、部分 UWP 可用）
    • 明显减少后台 CPU 占用
    • 游戏响应更快，但日常办公功能保留

AME Official:

  • 类型：极端去微软化配置 + 安装向导（Playbook 脚本是执行方案的一部分）
  • 核心特点：
    • 删除 Microsoft Store、Xbox、Cortana、Edge、OneDrive 等所有在线依赖
    • 阻断 Microsoft 登录体系（只支持本地账户）
    • 完全关闭 Windows Update（需要手动更新补丁）
    • 删除遥测、DRM 相关组件，替换部分 DLL 以绕过依赖
  • 安装后主要变化：
    • 系统极度干净、后台进程非常少（免打扰）
    • 丧失部分 Windows 生态功能（不能直接使用商店 / Xbox Game Pass / MSA 同步）
    • 非常适合隐私优先的专用机或嵌入式用途，但不适合普通用户
名称 来源与性质 定位/目标
Atlas OS 国外开源优化项目（偏游戏玩家圈），提供 PowerShell/批处理 Playbook 脚本 为游戏性能优化的精简 Windows，减少延迟、输入延时，降低系统占用
ReviOS 国外 Revios 团队维护的精简版 Windows（基于 LTSC/Pro），预设好的优化 ISO 在保持功能可用的前提下，移除遥测、广告、服务冗余，让系统更流畅稳定
AME (Ameliorated Windows) Ameliorated 小组制作的“去微软化”方案，有自己的 AME Wizard 作为安装执行器 完全删除 Microsoft 账户体系、大量内置组件和在线依赖，隐私优先

安装流程

打开下载好的AME Wizard

首先需要将你已经下载好的Playbook文件拖进左上角的虚线框中才会出现Revi OS的标志，入上图所示。

根据提示关闭所有的Windows Defender防护

功能选择

浏览器替换

可以选择不替换原有的浏览器

Defender移除

这里选择不移除

功能移除

1.移除Edge浏览器

2.移除OneDrive

3.移除Recall AI (仅在Windows 11有提供)

此处根据自己需要选择，本人建议一个都不要动，你要明白，这里没有后悔药（除非你创建了还原点）

UWP软件移除

1.移除照片程序 （建议保留）

2.移除Dev home (推荐移除)

3.移除Xbox （建议保留）

4.移除手机连接 （推荐移除，基本上用不到）

常规修改

1.应用Revi OS官方壁纸 （随意）

2.开启暗色模式 （随意）

3.将Windows 11右键菜单修改为Windows 10样式 （后面可再次修改）

4.启用旧版本的照片查看器 （不建议开启）

  1. 修改资源管理器的图标 （不建议小白开启）
  2. 移除开始菜单的九宫格快捷方式 （不建议移除）
  3. 修复Revi OS 设置 (建议开启)
  4. 关闭Windows自动更新 （根据个人情况开启）

等待重启
